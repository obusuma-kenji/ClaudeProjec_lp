GitHub Pagesで公開する場合は、上記ファイルを同一リポジトリのルートに配置してください。
`index.html` がLPとして表示され、同梱ファイルのダウンロードリンクが機能します。

## 販売・配布時の案内文

このパッケージは、Claude Projectを活用して士業業務を8つの専門AIアシスタントに分解し、
セミナー企画、提案書作成、情報発信、法改正対応、顧客対応、企画立案、分析、業務管理までを
体系的に運用できる実践キットです。

Claude ProまたはMax、Googleアカウントがあれば、セットアップ手順書に沿って約45分で環境構築できます。
